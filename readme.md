<p>

虽然有的API暂时来说比较新，兼容性还比较差，不太可能用到我们现在的项目中。不过了解一下，也许在合适的时机去使用
这些API能够极大的<strong>提升我们的开发效率</strong>、<strong>改善用户体验</strong>、<strong>优化页
面</strong>、<strong>丰富网页开发的可玩性</strong>等等。
</p>
